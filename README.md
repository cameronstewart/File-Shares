# File-Shares
File Share Analysis Tools

## FileNamePath Export with Hash.ps1
PowerShell Script to extract data for all files in a directory and export to csv.

Data includes 
* 'Path'
* 'Directory'
* 'Name'
* 'CreationTime'
* 'LastAccessTime'
* 'LastWriteTime'
* 'Extension'
* 'BaseName'
* 'B'
* 'MD5'
* 'SHA1'
